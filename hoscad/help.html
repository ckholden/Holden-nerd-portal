Setting a unit to AV automatically clears its incident assignment. Going OOS prompts for a reason.
AV FORCE: If a unit has an active incident, typing AV <UNIT> will warn you. Add FORCE to override: AV EMS1 FORCE. Force usage is logged in the audit trail.

PRIORITY <INC> <PRI-N>Update incident priority
e.g. PRIORITY 0023 PRI-1
STATSLive board summary: active incidents, units by status, longest open incident

Incident Types (NC command)

TYPE format: CAT-NATURE-DETERMINANT (e.g. MED-CARDIAC-CHARLIE)
Categories: MED, TRAUMA, FIRE, RESCUE, HAZMAT, OTHER
EMD Determinants: DELTA (critical/red), CHARLIE (serious/orange), BRAVO (moderate/yellow), ALPHA (routine/blue)
Priority: PRI-1 (critical), PRI-2 (urgent), PRI-3 (routine), PRI-4 (low)
Mutual Aid: Add "MA" anywhere in note to flag as mutual aid [MA]
Callback: Add [CB:PHONE] in note for callback number
Example: NC ST CHARLES; MA 67 YOF CARDIAC [CB:5415550123]; MED-CARDIAC-CHARLIE; PRI-1
Example: NC BEND RURAL; MVC WITH ENTRAPMENT; TRAUMA-MVA-DELTA; PRI-2
Example: NC MADRAS ED (minimal — just destination)
